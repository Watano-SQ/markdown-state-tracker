### 有点用的指令（cv 多了怕忘）
**更新系统**

```bash
sudo apt update && sudo apt upgrade -y
```

---

**新建账户（如 uaic）**

```bash
sudo adduser uaic
```

**赋予 sudo 权限（可选）**

```bash
sudo usermod -aG sudo uaic
```

**创建项目目录**

```bash
sudo mkdir -p /opt/uaic
sudo chown uaic:uaic /opt/uaic
```

---

**上传项目代码**

用 SFTP/WinSCP/Xftp 或 `scp` 命令将本地项目文件上传到 `/opt/uaic` 目录。

---

**配置国内 Docker 镜像加速器（未成功）**

1. 申请阿里云加速器地址：[阿里云容器镜像服务](https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors)
2. 编辑 `/etc/docker/daemon.json`，内容如下：

```json
{
  "registry-mirrors": ["https://你的ID.mirror.aliyuncs.com"]
}
```

3. 重启 Docker：

```bash
sudo systemctl restart docker
```

---

**启动后端服务**

1. 切换到 uaic 用户并进入项目目录：

```bash
sudo su - uaic
cd /opt/uaic
```

2. 启动服务：

```bash
docker-compose up -d
```

3. 检查服务状态：

```bash
docker ps
```

---

**安装 Nginx 并配置代理**

**安装 Nginx**

```bash
sudo apt install -y nginx
```

1. **配置 Nginx 代理到 **`/uaic`** 路径**  
编辑 `/etc/nginx/sites-available/uaic`，内容如下：

```plain
server {
    listen 80;
    server_name your-domain.com; # 或服务器公网IP

    location /uaic/ {
        proxy_pass http://127.0.0.1:8001/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

**启用配置并重启 Nginx：**

```bash
sudo ln -s /etc/nginx/sites-available/uaic /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

---

**开放安全组端口**

在阿里云控制台安全组设置中，开放 80 端口（HTTP），如需 HTTPS 也开放 443。

---

**访问测试**

+ 浏览器访问 `http://你的服务器公网IP/uaic` 或绑定的域名 `/uaic` 路径，能看到你的服务即部署成功。

---

**补充说明：**

+ 项目目录规范为 `/opt/uaic`，权限归属 `uaic` 用户。
+ Docker Compose 文件和代码均放在 `/opt/uaic` 下。
+ Nginx 代理路径为 `/uaic/`，如需前端适配请修改 API 地址和静态资源路径。
+ Docker Compose 已设置 `restart: unless-stopped`，服务会自动守护，无需额外 systemd 配置。

没错就是发现了这玩意，发现我们做了两周苦力（苍天呐，为什么当时没多研究一下

[https://github.com/unclecode/crawl4ai](https://github.com/unclecode/crawl4ai)

`crawl4ai` 是一个面向 AI 时代的通用爬虫框架，主要功能包括：

+ **动态网页抓取**：自动处理 JavaScript 渲染，支持 Playwright 浏览器自动化，能采集动态加载内容。
+ **多种数据抽取方式**：支持 XPath、CSS 选择器、正则表达式等方式提取网页结构化数据。
+ **配置化爬取**：通过 YAML/JSON 配置文件定义抓取规则，无需编写复杂代码即可采集不同网站。
+ **异步与并发**：支持异步爬取和多任务并发，提高采集效率。
+ **内容清洗与结构化**：自动清理网页噪音，输出结构化内容（如正文、标题、媒体、链接等）。
+ **媒体资源下载**：可自动下载图片、视频等媒体文件。
+ **自动化测试与诊断**：内置环境检测、爬虫健康检查工具（如 crawl4ai-doctor）。
+ **多格式输出**：支持 JSON、Markdown、HTML 等多种格式的数据输出。
+ **扩展性强**：可集成自定义处理逻辑，适配多种数据存储和分析场景。

适用于新闻、论坛、电商、学术等多种网站的数据采集任务。

其默认使用 Playwright，并且根据一些内置的算法和模型，自动清洗爬虫数据，很适合写通用型爬虫

### 通用型爬虫
写了个爬本科生院的通用型爬虫

```python
from crawl4ai import *
import json
import asyncio
from bs4 import BeautifulSoup
import aiofiles

async def crawl(domain, start_urls, max_depth=2, max_pages=100):
    q = asyncio.Queue()
    visited = set()
    count = [0]

    # 不可爬取的后缀名列表
    skip_suffixes = ['.pdf', '.doc', '.docx', '.xls', '.xlsx', '.ppt', '.pptx', '.zip', '.rar', '.7z', '.tar', '.gz', '.jpg', '.jpeg', '.png', '.gif', '.mp4', '.avi', '.mov']
    def can_crawl(link):
        return not any(link.lower().endswith(suffix) for suffix in skip_suffixes)

    for url in start_urls:
        if can_crawl(url):
            await q.put((url, 0))

    async def work():
        async with AsyncWebCrawler() as crawler:
            while not q.empty() and count[0] < max_pages:
                url, depth = await q.get()
                if url in visited or depth > max_depth or not url.startswith(domain):
                    continue
                visited.add(url)
                try:
                    result = await crawler.arun(url)
                    title = str(result.metadata.get("title", ""))
                    description = str(result.metadata.get("description", ""))
                    links = list(result.links.get("internal", []))
                    if not title:
                        print(f"Skipping {url} due to missing title")
                    elif not description:
                        print(f"Skipping {url} due to missing description")
                    else:
                        async with aiofiles.open(f".\\data\\{title}.md", "w", encoding="utf-8") as f:
                            await f.write(f"# {title}\n\nSource: {url}\n\n{result.markdown}")
                        print(f"Saved {url} as {title}.md")
                        count[0] += 1
                        print(count[0])
                    for link in links:
                        if link["href"] not in visited and can_crawl(link["href"]):
                            await q.put((link["href"], depth + 1))
                except Exception as e:
                    print(f"Error crawling {url}: {e}")

    concurrency = 10
    tasks = [asyncio.create_task(work()) for _ in range(concurrency)]
    await asyncio.gather(*tasks)

if __name__ == "__main__":
    domain = "https://jw.nju.edu.cn/"
    start_urls = [domain]
    asyncio.run(crawl(domain, start_urls, max_depth=286, max_pages=10000))
```
