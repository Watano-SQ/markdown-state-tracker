## 💡 关键概念

### 虚拟环境（Conda）
- **为什么需要**: 隔离项目依赖，避免冲突
- **如何创建**: `conda create -n markdown_tracker python=3.11 -y`
- **如何激活**: `conda activate markdown_tracker`

### 快速测试 vs 完整测试
- **快速测试** (`--skip-extraction`): 跳过 LLM，验证基础流程（3 秒）
- **完整测试** (`python main.py`): 包含 LLM 抽取（20 秒）

### API Key 配置
- **在哪里**: `.env` 文件
- **从哪获取**: OpenAI 网站
- **何时需要**: 完整测试时需要
